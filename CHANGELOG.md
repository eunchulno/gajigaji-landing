# Changelog

모든 주요 변경 사항이 이 파일에 기록됩니다.

## [1.2.0] - 2026-01-22

### Added
- **Quill WYSIWYG 에디터** 적용 (Markdig 마크다운 에디터 대체)
  - 이미지 붙여넣기/드래그앤드롭 지원 (Base64 임베딩)
  - 완전한 오프라인 지원 (CDN 의존성 제거)
  - 툴바: 제목, 굵게, 기울임, 밑줄, 취소선, 목록, 인용, 코드블록, 링크, 이미지, 색상
- 메모 패널 토글 기능 (같은 할일 클릭 시 패널 열기/닫기)
- 제목 입력 후 Enter 키로 Quill 에디터 포커스 이동

### Changed
- 앱 창 크기 조정 (800x600 → 1100x700)
- 메모 패널 레이아웃 개선
  - 패널 닫힐 때 중앙 영역이 전체 확장되도록 수정
  - 패널 크기 조절 가능 (250~600px)
- Ctrl+Z 키 처리 개선
  - 메모 에디터 활성 시 Quill의 실행취소 동작
  - 앱 영역 활성 시 작업 실행취소 동작
- 메모 에디터 테마가 앱 테마와 자동 동기화

### Fixed
- 제목 편집 시 메모 패널이 닫히는 버그 수정
- WebView2 포커스 감지 개선

### Removed
- Markdig 패키지 제거 (Quill로 대체)

## [1.1.1] - 2026-01-22

### Added
- Empty state에 앱 사용법 가이드 추가
  - 빠른 시작 가이드 (키보드 단축키 안내)
  - 주요 기능 소개 (해시태그, 중요 표시, 날짜, 반복)
  - 팁 섹션 (자연어 입력, 사이드바, 펫 시스템)
- CLAUDE.md 프로젝트 문서 추가

### Changed
- TaskListView.xaml의 empty state UI 개선

## [1.1.0] - 이전 버전

### Features
- 할일 관리 (CRUD)
- 서브태스크 지원
- 해시태그 시스템
- 프로젝트 그룹화
- 반복 설정 (일/주/월/년)
- 알림 기능
- 펫 캐릭터 시스템
- 통계 및 스트릭
- 다크/라이트 테마
- Undo/Redo
- 자연어 입력 파싱
- 시스템 트레이 지원
